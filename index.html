PDF FORMAT GUIDE
For best auto-detection, your PDF should have columns like:
Key ID Type Make Model Year VIN Status Date
• Key Type — use "Aftermarket", "OEM Recase", or "OEM Refurb" (partial matches like "aftermarket", "recase", "refurb" also work)
• Status — use "Available", "Sold", or "Damaged"
• VIN — optional 17-character vehicle ID
• Barcodes — if your PDF has barcodes starting with AM-, OR-, OF-, those will auto-classify the key type
💡 Tip: If your PDF doesn't parse perfectly, you can review and edit each detected row before importing. Any unrecognized key type will be flagged for manual selection.
